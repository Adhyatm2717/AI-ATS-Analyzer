# AI-ATS-Analyzer

A premium, modern applicant tracking system simulation tool designed to help candidates optimize their resumes against job descriptions.

## Features
- **Modern SaaS UI**: Glassmorphism, dynamic mesh gradients, and smooth layout interactions.
- **Deep ATS Simulation**: Real-time mock analysis of resumes for keywords, formatting, and relevance.
- **Visualized Metrics**: SVG-based score circles and categorized skill breakdown (matching vs. missing).
- **Responsive Layout**: Works seamlessly across both desktop and mobile views.

## Technologies Used
- HTML5
- CSS3 (Vanilla, no frameworks)
- JavaScript (Fetch API, DOM manipulation, IntersectionObserver)

## Usage
Simply drop in your resume, paste a target job description, and the built-in logic simulates an ATS parsing flow.
